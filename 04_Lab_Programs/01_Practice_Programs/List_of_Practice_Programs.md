# Practice Programs

This repository contains a collection of beginner-friendly Practice Python programs written by **Rushikesh Padaki**. Each program includes:

- Title, Author, and Date  
- Problem Description and Algorithm  
- Time and Space Complexity  
- Sample Input/Output  

---

## List of Programs

| S.No | File Name                             | Title                                          | Description                                                                                   |
|------|----------------------------------------|------------------------------------------------|-----------------------------------------------------------------------------------------------|
